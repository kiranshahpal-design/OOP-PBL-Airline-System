\# Airline Reservation \& Flight Management System



\## How to Build and Run

1\. Open terminal in this folder.

2\. Compile using: g++ -std=c++17 \*.cpp -o airline\_system

3\. Run using: ./airline\_system



\## Features Included

\- File Persistence (Reads from data.txt)

\- 3 Flight Types (Domestic, International, Charter)

\- 3 Passenger Types (Economy, Business, First Class)
